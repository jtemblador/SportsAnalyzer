TECHNICAL INSIGHTS
================================================================================

Why V4 Succeeded:
  1. Vegas lines predict game script (trailing teams pass more)
  2. Team totals predict volume opportunity
  3. Position-specific tuning captures edge cases
  4. Built on V2's solid variance/trend foundation
  5. Game context > player efficiency metrics

Why V3 Failed:
  1. EPA derived from stats already in V2 (redundant)
  2. Efficiency metrics don't predict volume
  3. CPOE/PACR/WOPR correlated with existing features
  4. 57 features on 28K records = overfitting
  5. Domain knowledge mislead us

Why WR Regressed:
  1. Vegas predicts TEAM totals, not individual WR distribution
  2. 3+ WRs split targets unpredictably
  3. Defensive schemes (shadow coverage) vary
  4. Different validation weeks may explain difference
  5. WR needs target distribution model

Best Practices Learned:
  - Version everything (reproducibility)
  - Validate on held-out test set (catch regressions)
  - Document failures (learn what doesn't work)
  - Simpler is often better (V2 > V3)
  - Context matters more than analytics

================================================================================
TECHNICAL INSIGHTS
================================================================================

WHY V3 FAILED:

1. Feature Redundancy
   - EPA is derived from yards/TDs already in V2 rolling averages
   - CPOE correlates with passing_yards variance
   - Model already learned these patterns from simpler features

2. Efficiency ≠ Fantasy Points
   - PACR, RACR measure efficiency (yards per target)
   - Fantasy needs volume (total targets)
   - A player can be efficient but low-volume → low points

3. Position Decay Didn't Matter
   - QB: 0.90 vs V2's 0.85 made no difference
   - TE: 0.80 helped marginally (+1.4%)
   - Variance features already capture position volatility

4. Training Data Limits
   - QB: only 2893 records
   - K: only 2592 records
   - Complex features → overfitting on small datasets

WHY V2 SUCCEEDED:

1. Variance features capture boom/bust tendency
2. Trend features identify hot/cold streaks
3. Decay 0.85 weights recent games appropriately
4. Simple features generalize better on limited data

   Decision: Merge Vegas team lines into feature set during training
   Rationale: Game context (spread, totals) predicts volume/opportunity
   Implementation: 8 new features added to existing 42 V2 features
   Impact: Models learn patterns like "Underdogs pass +40%, favorites run +30%"

2. NEUTRAL DEFAULTS FOR HISTORICAL DATA
   Decision: Use neutral values (spread=0, total=45) for 2020-2024
   Rationale: No historical betting lines available, need consistent feature set
   Implementation: add_vegas_features() returns defaults if no Vegas data
   Impact: Historical data still trains models (without game context)

3. POSITION-SPECIFIC VOLUME INDICES
   Decision: Create QB-specific and RB-specific volume predictors from Vegas
   Rationale: Different positions respond differently to game script
   Implementation:
     - qb_volume_index: Higher when trailing (underdogs pass more)
     - rb_volume_index: Higher when favored (favorites run more)
   Impact: Models can learn position-specific game script effects

4. 50-FEATURE ARCHITECTURE
   Decision: Keep all V2 features, add Vegas on top
   Rationale: V2 works well (4.66 MAE), Vegas adds new dimension
   Implementation: 42 V2 (variance, trends) + 8 Vegas (context)
   Impact: Best of both worlds - player patterns + game context

TECHNICAL NOTES
================================================================================

Vegas Feature Calculations:

game_script_index:
  = (-point_spread × 0.3) + ((over_under - 45) × 0.1)
  Interpretation:
    - Positive = pass-heavy expected (trailing or high-scoring)
    - Negative = run-heavy expected (leading or low-scoring)

qb_volume_index:
  = max(0, point_spread) × 0.5 + (team_implied_total - 20) × 0.3
  Interpretation:
    - Higher when underdog (trailing teams pass more)
    - Higher when high team total (more offensive plays)

rb_volume_index:
  = max(0, -point_spread) × 0.5 + (team_implied_total - 20) × 0.2
  Interpretation:
    - Higher when favorite (leading teams run more)
    - Higher when high team total (more offensive plays)

These formulas are heuristics based on NFL research showing:
  - Trailing teams: +40% pass attempts
  - Leading teams: +30% rush attempts
  - High totals: +15% plays overall

Goal
-----
Pull all 3 PFR Advanced Stats datasets from nflreadpy for seasons 2018-2025.
These give us pressure rates, bad throw rates, yards before/after contact,
broken tackles, drop rates, and passer rating when targeted — quality and
context metrics beyond box scores. Combined into one task since all 3 use
the same function with different stat_type.

Approach
--------
- Used nfl.load_pfr_advstats(seasons, stat_type) for pass/rush/rec.
- Single PFRAdvStatsFetcher class mirrors NextGenStatsFetcher pattern.
- One Parquet file per season per type: pfr_{type}_{season}.parquet (24 files).
- pfr_player_id is the join key — PFR format, same as snap counts.

Lessons Learned
---------------
- PFR data uses pfr_player_id (same format as snap counts, e.g. "MahoPa00").
  Two datasets now need PFR→GSIS ID mapping during feature engineering:
  snap_counts and pfr_advstats. load_players() has both IDs.
- Skipped 'def' stat_type — defensive player stats aren't relevant for
  fantasy scoring predictions of offensive skill positions.

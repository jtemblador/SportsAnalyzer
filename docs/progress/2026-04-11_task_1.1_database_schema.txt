Goal
-----
Create the nfl_predictions PostgreSQL database with all 14 tables matching
our 13 datasets + auto-generated ID column. Schema auto-generated from
Parquet files to avoid hand-typing 600+ column definitions.

Approach
--------
- PostgreSQL 15 already installed and running on the system.
- Set password 'nfl' for user j0e.
- Built generate_schema.py that reads each Parquet file, maps pandas dtypes
  to PostgreSQL types (int32->INTEGER, float64->DOUBLE PRECISION, object->TEXT),
  and outputs CREATE TABLE statements.
- All column names quoted with double quotes to handle SQL reserved words.
- Each table gets an auto-generated `id SERIAL PRIMARY KEY` column.
- 16 indexes created for common query patterns.

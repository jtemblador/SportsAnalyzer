   - On track for competitive accuracy

================================================================================
WHAT NEEDS WORK
================================================================================

1. QB Predictions - Both V1 and V2 struggle (R² near 0)
   - QB performance has high variance (5-40 point range)
   - Need position-specific hyperparameters
   - EPA features might help (passing_epa, cpoe)
   - Opponent defense rank not predictive enough

2. RB Slight Regression - V2 worse by 0.19 MAE
   - Possible overfitting on variance features
   - RBs with high variance (boom/bust) may be over-penalized
   - Consider: EPA, snap share, red zone carries for V3

3. POB Model Pessimism - 38% beat rate (should be ~50%)
   - Model predicts players will underperform baseline too often
   - Need to recalibrate or adjust baseline calculation
   - Could use quantile regression instead of binary classification

4. Boom/Bust Games - Still can't predict outliers
   - J.Taylor 49.6 points (predicted 19.2) - off by 30 points
   - J.Gibbs 55.4 points (predicted 16.8) - off by 38 points
   - These are 3+ standard deviation events - inherently unpredictable
   - Variance features help but can't eliminate surprise performances
